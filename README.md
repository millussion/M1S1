# Basic Inventory System / M1S1

## Description

This console program allows the user to register a product by entering its name, price and amount. The system calculates the total cost automatically and displays the stored information.

## Structure

The project is organized into three modules:

* **main.py** — controls the program execution
* **products.py** — handles product creation and storage using a list of dictionary
* **view.py** — displays the saved product data (name, price, amount and total)

## Logic

The program uses two loops and exception handling to validate the numeric input. Each product is stored in a dictionary containing the name, price, quantity, and total. A loop is used to iterate over the list of products and print the formatted results.
